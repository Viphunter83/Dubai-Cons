# Dubai Cons Frontend

React + Vite frontend for Dubai Cons AI Suite MVP.

## Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Available Pages

- **/** - Home page with features
- **/projects** - View and manage projects
- **/clients** - View and manage clients
- **/design** - AI design generator

## API Integration

Frontend connects to the backend API running on `http://localhost:8000`

## Development

The frontend is configured to proxy API requests to the backend during development.
